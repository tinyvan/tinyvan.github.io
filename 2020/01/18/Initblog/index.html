使用Hexo + Nexmoe构建博客

2020年01月18日 413 字 大概 1 分钟

去年在Github探索中发现了Hexo库,打断了我徒手(手写html)的想法

为凸显个人风格(粉红的少女心),决定以Nexmoe作为模板

开始

hexo

前置依赖:

npm,node,git

Ubuntu 可以sudo apt install npm node git搞定,其他发行版用各自的软件包管理娘

至于windows,一个一个网上查吧(貌似直接安装git和node.js就可以了)

为了方便以后维护,单独创建一个安全(防丢防删防找不到)的文件夹,笔者常用生产系统是Ubuntu,就放在~/桌面文件夹好了(~/文档放代码)

启动终端并定位到上文的文件夹,

Linux:npm install hexo-cli -g

笔者操作时报错:ERR! cb() never called!
解决方法:[本段引用自cnblog
1.清除npm缓存cache clean -f
2.安装最新的node helper npm install n -g
3.助手安装最新稳定版node n stable
再次安装hexo npm install hexo-cli -g

安装完毕后运行hexo Init blog(blog 是你的博客文件夹,可换成你喜欢的名字)

进入blog文件夹cd blog

启动bloghexo s

出现01:55:11.102 INFO Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.则配置成功,可以打开浏览器访问localhost:4000​

在控制台Ctrl+C可以关闭

在config.yml中可以对hexo进行配置
